Project Overview
----------------
1. naver.py: A script that crawls data from Naver Blog, Cafe, and News by searching for specific keywords.
2. someTrend.py: A script that performs related word analysis or sentiment analysis on a given keyword from the SomeTrend website and saves the results as a CSV file.

Required Libraries
------------------
- selenium: Web automation tool
- beautifulsoup4: Library for parsing HTML
- pandas: Used for creating DataFrames and saving data as CSV files
- chromedriver_autoinstaller: Automatically installs the appropriate ChromeDriver version
- calendar: Built-in Python library for handling calendar-related functions

How to Install Libraries
------------------------
pip install selenium beautifulsoup4 pandas chromedriver-autoinstaller

프로젝트 개요
1. 네이버.py: 네이버 블로그, 카페, 뉴스에서 특정 키워드를 검색하여 관련 데이터를 크롤링하는 스크립트
2. 썸트렌드.py: 썸트렌드 웹사이트에서 특정 키워드에 대한 연관어 분석 또는 긍·부정 분석을 수행하고 결과를 CSV 파일로 저장하는 스크립트


필수 라이브러리
selenium: 웹 자동화 도구
beautifulsoup4: HTML 파싱을 위한 라이브러리
pandas: 데이터프레임 생성 및 CSV 파일 저장
chromedriver_autoinstaller: 크롬 드라이버 자동 설치 도구
calendar: 달력 관련 함수 (Python 기본 라이브러리)


라이브러리 설치 방법
pip install selenium beautifulsoup4 pandas chromedriver-autoinstaller
